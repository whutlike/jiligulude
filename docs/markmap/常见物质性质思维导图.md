## 氧气

```markmap
## 物理性质

- 无色无味气体
- 密度比空气大
- 不易溶于水

## 化学性质

### 供给呼吸

### 支持燃烧

- 氧气能使带火星的木条复燃
- 红磷燃烧

  - 现象

    - 空气：放热，产生大量白烟
  - 化学方程式

    - $\ce{4P + 5O2}\xlongequal{点燃}\ce{2P2O5}$
- 木炭燃烧

  - 现象

    - 空气：发出红光，放热，生成使澄清石灰水变浑浊的气体
    - 氧气：发出白光，放热，生成使澄清石灰水变浑浊的气体
  - 化学方程式

    - $\ce{C +O2}\xlongequal{点燃}\ce{CO2}$
- 硫燃烧

  - 现象

    - 空气：发出微弱的淡蓝色火焰，放热，生成有刺激性气味的气体
    - 氧气：发出蓝紫色火焰，放热，生成有刺激性气味的气体
  - 化学方程式

    - $\ce{S + O2}\xlongequal{点燃}\ce{SO2}$
- 铁丝燃烧

  - 现象

    - 空气：铁丝红热，不能燃烧
    - 氧气：剧烈燃烧，火星四射，放热，生成黑色固体
  - 化学方程式

    - $\ce{3Fe + 2O2}\xlongequal{点燃}\ce{Fe3O4}$
- 镁条燃烧

  - 现象

    - 空气：发出耀眼的白光，产生白烟，放热，生成白色固体
  - 化学方程式

    - $\ce{2Mg + O2}\xlongequal{点燃}\ce{2MgO}$

## 用途

- 动植物呼吸、医疗急救（供给呼吸）
- 炼钢、气割、气焊、航天航空（支持燃烧）
```

## 二氧化碳

```markmap
## 物理性质

- 无色无味的气体
- 密度比空气大
- 能溶于水

## 化学性质

### 1.不能燃烧也不支持燃烧

### 2.与水反应

- $\ce{CO2 + H2O}\xlongequal{}\ce{H2CO3}$
- $\ce{H2CO3}\xlongequal{}\ce{CO2 ^ + H2O}$（碳酸不稳定）

### 3.与碱反应

- $\ce{CO2 + Ca(OH)2}\xlongequal{}\ce{CaCO3 v + H2O}$
- $\ce{CO2 + 2NaOH}\xlongequal{}\ce{Na2CO3 + H2O}$

### 4.与碳反应

- $\ce{C + CO2}\xlongequal{高温}\ce{2CO}$

## 用途

- 光合作用的原料
- 灭火
- 化工产品的原料
- 气体肥料
- 干冰（固态二氧化碳）

  - 制冷剂
  - 人工降雨

## 制取

### 实验室制取

#### 药品

- 大理石（或石灰石）与稀盐酸

#### 原理

- $\ce{CaCO3 + 2HCl}\xlongequal{}\ce{CaCl2 + H2O + CO2 ^}$

#### 发生装置

- 固液不加热型

#### 收集装置

- 向上排空气法

#### 检验

- 将气体通入澄清的石灰水，若澄清的石灰水变浑浊，则证明气体是二氧化碳

#### 验满

- 将燃着的木条放在集气瓶口，若木条熄灭，则证明已集满

### 工业制取

- $\ce{CaCO3}\xlongequal{高温}\ce{CaO + CO2 ^}$
```

## 金属

```markmap
## 物理性质

### 共性

- 常温下为固体（汞为液体）
- 大多为银白色（铜：紫红色、金：黄色、铁粉：黑色）
- 有光泽、熔点高、硬度大
- 有良好的导电性、导热性、延展性

### 特性

- 熔点最低的金属-汞（Hg）
- 地壳中含量最高的金属-铝（Al）
- 人体含量最高的金属-钙（Ca）
- 年产量最高的金属-铁（Fe）

### 用途

- 作金属饰品-有金属光泽
- 制作导线-导电性
- 制作炊具-导热性
- 能压成薄片、拉成丝、能弯曲-延展性

## 化学性质

### 1.金属+氧气→金属氧化物

- $\ce{2Cu + O2}\xlongequal{\Delta}\ce{2CuO}$
- $\ce{4Al + 3O2}\xlongequal{}\ce{2Al2O3}$
- $\ce{3Fe + 2O2}\xlongequal{点燃}\ce{Fe3O4}$

### 2.（活泼）金属+酸→盐+氢气（**置换反应**）

- $\ce{Zn + H2SO4}\xlongequal{}\ce{ZnSO4 + H2 ^}$
- $\ce{Fe + 2HCl}\xlongequal{}\ce{FeCl2 + H2 ^}$

### 3.金属+盐→盐+金属（**置换反应**）

- $\ce{2Al + 3CuSO4}\xlongequal{}\ce{Al2(SO4)3 + 3Cu}$
- $\ce{Cu + 2AgNO3}\xlongequal{}\ce{Cu(NO3)2 + 2Ag}$
```

## 酸

```markmap
## 盐酸

### 特性

- 无色有刺激性气味的液体

- 挥发性：打开试剂瓶口能观察到白**雾**

### 用途

- 用于金属表面除锈
- 胃酸的主要成份，可以帮助消化

## 硫酸

### 特性

- 无色无味的液体

- 强烈的腐蚀性（脱水性）

- 吸水性

### 用途

- 金属除锈

- 做干燥剂（吸水性）

## 酸的化学性质

### 1.与指示剂作用

- 酸能使紫色石蕊溶液变红，不能使无色酚酞溶液变色

### 2.酸+（活泼）金属→盐+氢气（**置换反应**）

- $\ce{Zn + H2SO4}\xlongequal{}\ce{ZnSO4 + H2 ^}$
- $\ce{Fe + 2HCl}\xlongequal{}\ce{FeCl2 + H2 ^}$

### 3.酸+金属氧化物→盐+水（**复分解反应**）

- $\ce{Fe2O3 + 6HCl}\xlongequal{}\ce{2FeCl3 + 3H2O}$
- $\ce{Fe2O3 + 3H2SO4}\xlongequal{}\ce{Fe2(SO4)3 + 3H2O}$

### 4.酸+碱→盐+水（**复分解反应**）

- $\ce{NaOH + HCl}\xlongequal{}\ce{NaCl + H2O}$
- $\ce{H2SO4 + 2NaOH}\xlongequal{}\ce{Na2SO4 + 2H2O}$

### 5.酸+盐→盐+酸（**复分解反应**）

- $\ce{2HCl + Na2CO3}\xlongequal{}\ce{2NaCl + H2O + CO2 ^}$
- $\ce{HCl + AgNO3}\xlongequal{}\ce{AgCl v + HNO3}$
- $\ce{H2SO4 + BaCl2}\xlongequal{}\ce{BaSO4 v + 2HCl}$
```
